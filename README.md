## 소개

> 티메(Timé) 는 사용자의 집중력 데이터를 기반으로 상대적인 시간 흐름을 체감할 수 있도록 돕는 macOS 앱입니다. 소개 생각해보기, 무가 쓴 거 뺏기

[🔗 앱스토어/웹 링크](https://example.com)

## 주요 기능

- createML, linear regression을 통해 사용자 집중력 점수 제공
- 집중력 점수에 따라 상대적인 시간 흐름 표현
